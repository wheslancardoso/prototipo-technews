TechNews – Protótipo Funcional (MVP)

Primeira versão navegável para apresentação: frontend simples + backend Spring Boot + BD H2.

Funcionalidades Essenciais

  • Preview da Newsletter (conteúdo HTML gerado pelo backend)
  • Abrir Preview bruto (endpoint)
  • Console H2 (ambiente de desenvolvimento)

Rotas de Backend

  • GET /api/newsletter/email/preview: retorna HTML da newsletter
  • GET /api/newsletter/email/smoke: teste de envio (Mailgun/SMTP)

Estado do Ambiente

Perfil ativo: dev (H2 em memória). Para envio real, configure Mailgun no .env.
